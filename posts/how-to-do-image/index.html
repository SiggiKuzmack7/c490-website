Adding images to your Hugo post

Create a new top-level folder in your site repo. By “top-level” I mean that it should be a sibling of archetypes and content. Name the new folder static. Put an image in that folder.

To link your image from a post, use code like this:

![Here is frog](/frog.jpg)
